# urp

Allows you to quickly convert textures to upr/lit format.  

Just place the exe file next to the textures that end in:
- BaseColor.png
- Normal.png
- Roughness.png
- Metallic.png
 
and run it. New textures for the urp/lit shader are created in the textures folder.

## build

```bash
cargo build --release
```
